
                    Social Media Analytics Dashboard

A frontend dashboard that shows social media performance data with interactive charts.

**Live Demo**
https://social-dashboard-two-orcin.vercel.app/

**Features**
- Overview cards showing followers, posts, engagement rate and likes
- Follower growth line chart
- Post engagement bar chart
- Audience age groups and top countries pie charts
- Switch between Instagram, Twitter and YouTube data
- Date range filter (Last 7 days, 30 days, 3 months, all time)
- Mobile responsive design

**Tech Stack**
- React 18 + Vite
- Recharts
- React Icons
- CSS

**How to run locally**
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open localhost:5173 in your browser
